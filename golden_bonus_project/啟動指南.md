## 🎯 啟動應用程式

### 方法一：使用 Streamlit（推薦）

```bash
cd golden_bonus_project
streamlit run main.py
```

應用程式會自動在瀏覽器中打開，通常是 `http://localhost:8501`

### 方法二：使用 Python 直接執行（僅測試）

```bash
cd golden_bonus_project
python3 main.py
```

## 📋 使用流程

1. **填寫企業數據**
   - 在左側 Sidebar 填寫：
     - 年度總營收（萬元）
     - 稅前淨利（萬元）
     - 符合發放資格人數
     - 平均月薪（元）

2. **調整決策槓桿**
   - **生存槓桿**：調整「公司安全氣囊 (%)」，觀察存活月數估算
   - **激勵槓桿**：選擇「人才投資策略」（留才優先/戰功優先/團隊優先）
   - **現實槓桿**：確認財務數據

3. **生成決策備忘錄**
   - 點擊「🚀 開始分析 / 生成草案」
   - 等待 AI 分析（約 5-10 秒）
   - 查看：
     - Metrics 儀表板（總獎金池、人均金額、平均月數）
     - 決策備忘錄（黃金三段式輸出）

4. **互動諮詢**
   - 使用快捷按鈕快速提問
   - 或在聊天框輸入自訂問題
   - AI 會基於您的數據提供建議

## ⚠️ 注意事項

1. **數據變動後必須重新生成**
   - 調整 Sidebar 數據後，系統會自動禁用聊天功能
   - 必須點擊「生成草案」更新分析後才能繼續聊天

2. **API 使用成本**
   - 每次生成報告和聊天都會呼叫 Gemini API
   - 請注意 API 使用量

3. **錯誤處理**
   - 如果 API Key 無效，會顯示錯誤訊息
   - 如果計算錯誤（如除以零），會顯示友善的錯誤提示

## 🔧 疑難排解

### 問題：ImportError: No module named 'xxx'

**解決方案**：
```bash
# 確認當前目錄
pwd  # 應該顯示 .../golden_bonus_project

# 重新安裝依賴
pip install -r requirements.txt
```

### 問題：API Key 錯誤

**解決方案**：
1. 檢查 `.env` 檔案是否存在
2. 確認 API Key 格式正確（沒有多餘空格）
3. 確認 API Key 有效（可在 Google AI Studio 測試）

### 問題：Streamlit 無法啟動

**解決方案**：
```bash
# 確認 Streamlit 已安裝
pip show streamlit

# 如果未安裝，執行
pip install streamlit
```
